1 - Rodar o comando composer install (ela baixa as dependências do projeto na pasta vendor)
2 - Para instalar as dependências de um projeto que possui javascript eu devo rodar o comando npm install
3 - Após rodar o npm install e se o projeto tiver webpack ou vite rodar o comando npm run build
4 - Criar uma cópia do arquivo .env.example e renomear apenas para .env
5 - Criar uma chave encriptada para o projeto com o comando php artisan key:generate
6 - Rodar as migrations para criar o banco de dados com o comando php artisan migrate
